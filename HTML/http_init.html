Web Foundation1

A web technology stack consists of three components:

  • HTML(HyperText Markup Language): structure & content
  • CSS(Cascading Style Sheet): style
  • JavaScript: behaviour logic

HTTP: HyperText Transfer Protocol (server-client request-response protocol)

Request methods:

  • GET: retrieve data from server
  • POST: send data to server

HTTP Request Structure:

  • Request Line
    • Request method
    • Request URL (with query strings)
    • Protocol version(e.g. HTTP/1.1 HTTP/2.0)
  • Request Header
    • Host (e.g. www.google.com)
    • User-Agent (e.g. browser- type,version...)
    • Accept (indicating the media type that the client can understand)
      • text/html ->HTML
      • application/json -> JSON
      • */* -> any type
  • Request Body
    • data to be sent to the server

HTTP Response Structure:

  • Status Line
    • Protocol version
    • Status code
      • 200:OK
      • 301: Moved permanently, need to redirect
      • 401: Unauthorized
      • 404: Not found
      • 503: Server Unavailable
  • Response header
    • Date
    • Content-Type and Encoding format
  • Response body
    • if content type is text/html, then it is HTML content
